Underscore 扩展方法

indexOf2

  • _.indexOf2(array, callback)

根据回调函数查找数组中的元素，并返回该元素的索引

_.indexOf2([{ id: 1 }, { id: 2 }], function(value){
    return value.id === 1;
});
// result: 0

safeInvoke

  • _.safeInvoke(context, method, params)

调用对象的某个方法，当该对象为空时，不会报错

_.safeInvoke(this, 'read', { id: 1 });
